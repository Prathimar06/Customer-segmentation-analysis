# Customer Segmentation Analysis

This project performs customer segmentation using clustering algorithms to group customers based on their behavior.

## Dataset
The dataset contains customer purchasing information used for segmentation.

## Methods Used
- K-Means Clustering
- DBSCAN
- Hierarchical Clustering

## Results
Clusters show distinct customer groups with different spending patterns.

## How to Run
1. Clone this repo
2. Install requirements
3. Run the Jupyter notebook
